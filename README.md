# 版权出海服务平台

专业的跨国知识产权法律服务平台，提供法律法规查询、典型案例分析、实务指引及在线律师咨询服务。

## 快速开始

1. 安装依赖：
   ```bash
   pnpm install
   ```

2. 运行开发服务器：
   ```bash
   pnpm dev
   ```

3. 构建项目：
   ```bash
   pnpm build
   ```

## 一键部署
